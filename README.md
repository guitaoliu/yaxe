# xjtu-grade

西安交通大学成绩查询排名计算爬虫

## 项目简介

今年暑期学校教务处突然开启了成绩排名可查询，听说之前也开放过后来关了就想写个爬虫把数据记录下来

目前只对有意义的数据进行显示，但实际还爬到了一些很难处理的数据，如果有大佬想继续研究的话可以取消代码中某些注释(笑

目前表格中的项目有

- 学年学期
- 课程名
- 课程类别
- 课程性质
- 学分
- 学时
- 成绩
- GPA
- 及格
- 超过百分比
- 平时成绩
- 期末成绩
- 平均分
- 最低分
- 最高分

## 使用指北

### 项目依赖

本项目依赖

- python 3

python包依赖
- selenium（请确保能够正常运行，包括配置webdriver等
- xlwt

### 运行

运行`spider/spider.py`后

在运行目录下生成`Grade.xls`成绩表格

Enjoy it~
